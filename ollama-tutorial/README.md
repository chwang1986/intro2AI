# Ollama 调用教程

理解大模型接口、基本原理与高级用法。

## 运行方式

```bash
bash start.sh
```

访问 http://localhost:5006

## 内容大纲

| 步骤 | 主题 | 内容 |
|------|------|------|
| Step 1 | 大模型基本原理 | Transformer 架构、注意力机制、Token、参数规模 |
| Step 2 | Ollama 部署与接口 | 本地安装、REST API 结构、模型拉取与管理 |
| Step 3 | 基础调用方式 | Python requests / ollama 库、生成与聊天接口 |
| Step 4 | 高级用法 | 流式输出、系统提示、温度/top_p、多轮对话 |
| Step 5 | 实际应用 | 文本生成、代码辅助、结构化输出 / JSON Mode |
