# 前端设计系统教程

## 目标
总结一套可复用的前端设计规范，面向未来所有教程和应用。

## 设计理念
- **简洁明快**：白色/浅灰背景 + 蓝色渐变强调，拒绝冗余装饰
- **科技感**：扁平化、精确对齐、明确层级、适度留白
- **可复用**：所有样式以 CSS 类形式沉淀，即拿即用
- **大厂经验**：参考 Ant Design、TDesign 等国内主流设计系统

## 资源依赖（均免费商用）
- 字体：阿里巴巴普惠体
- 图标：IconPark
- 图表：ECharts
- 插画：unDraw

## 启动
```bash
./start.sh        # 默认端口 5004
./start.sh 5005   # 指定端口
```

## 结构
- `static/css/design-tokens.css` — 核心设计令牌与组件类
- `templates/` — 教学页面（step1~step10 + practice）
- `examples/` — 各阶段独立示例文件

## 阶段大纲
1. 设计原则（对齐、对比、重复、亲密性）
2. 色彩系统（令牌、主色、辅色、功能色）
3. 字体系统（字体族、字号层级、字重）
4. 间距与布局（8px 网格、留白、容器、断点）
5. 基础组件（按钮、输入框、标签、徽章）
6. 容器组件（卡片、面板、模态框、导航）
7. 数据展示（表格、列表、空状态、加载）
8. 响应式设计（移动端适配、弹性布局）
9. 动画与交互（过渡曲线、微交互）
10. 页面模板（可直接套用的页面结构）
