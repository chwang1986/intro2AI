Step 9 动画与交互

过渡曲线、状态反馈、微交互原则

进阶

场景

页面不是静止的图片，用户会和它互动：点击按钮、填写表单、加载数据。动画和交互的作用是让这些变化更自然，给用户即时的反馈，让用户知道"系统收到了我的操作"。

原理

为什么需要动画反馈

  • 降低焦虑：点击提交后如果没有任何反应，用户会怀疑自己到底点没点上。一个加载动画就能缓解这种焦虑。
  • 引导注意力：新内容出现时带一个淡入动画，用户的视线会自然地被吸引过去。
  • 强化因果关系：按钮按下去时有一个下沉效果，用户会感觉到"是我按下去的"。

Ant Design 的动效规范

  • 出场动画：淡入 + 轻微上移，持续 0.2s。用于弹窗、提示、新内容。
  • 离场动画：淡出 + 轻微上移，持续 0.15s。比出场快，因为用户不想再看到它。
  • 状态过渡：颜色、边框、阴影的变化持续 0.15s ~ 0.25s。太快会闪眼，太慢会拖沓。

缓动曲线（Easing）

缓动曲线决定了动画的速度变化：

  • ease：默认，先加速后减速。适合大多数过渡效果。
  • ease-out：开始快，后面慢。适合出场动画（元素进入视野）。
  • ease-in：开始慢，后面快。适合离场动画（元素离开视野）。
  • linear：匀速。适合循环动画（加载旋转、进度条）。

要点

  1. 动画时间控制在 0.15s ~ 0.4s：低于 0.1s 人眼察觉不到，高于 0.5s 会感觉拖沓。
  2. 优先使用 transform 和 opacity：这两个属性不会触发浏览器的重排（reflow），性能最好。避免直接改变 width、height、top、left。
  3. 减少运动偏好（prefers-reduced-motion）：有些用户不喜欢动画（或因为晕动症）。用 @media (prefers-reduced-motion: reduce) 关闭非必要动画。
  4. hover 状态要即时反馈：鼠标移上去 0.15s 内变色，让用户知道这是可点击的。
  5. 加载动画要循环：旋转、跳动、骨架屏的流动，必须循环播放，直到加载完成。

常见错误

❌ 动画太长

一个弹窗用了 1 秒钟才完全显示，用户已经等不及了。入场动画 0.2s，离场 0.15s 是最佳实践。

❌ 滥用动画

每个元素都有动画，页面像个万花筒。动画应当是"调味品"，只在需要引起注意的地方使用。

❌ 没有考虑性能

用 JavaScript 定时器改变元素的 left/top 来实现动画，会导致每帧都重排，卡顿明显。应当用 CSS transition/animation + transform。

完整代码

/* 旋转加载 */
@keyframes spin {
  to { transform: rotate(360deg); }
}
.spinner {
  width: 20px;
  height: 20px;
  border: 2px solid var(--border-color);
  border-top-color: var(--color-primary);
  border-radius: 50%;
  animation: spin 0.8s linear infinite;
}

/* 淡入上移动画 */
@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(10px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
.fade-in-up {
  animation: fadeInUp 0.2s ease-out;
}

/* 按钮交互 */
.btn {
  transition: all 0.15s ease;
}
.btn:hover {
  opacity: 0.85;
}
.btn:active {
  transform: translateY(1px);
}

/* 减少运动偏好 */
@media (prefers-reduced-motion: reduce) {
  .spinner,
  .fade-in-up {
    animation: none;
  }
  .btn {
    transition: none;
  }
}

练习

练习 1：给项目中的所有按钮添加 hover 和 active 过渡效果，使用 transform 而不是改变 width/height。

练习 2：实现一个 toast 提示组件，出现时从上方滑入并淡入，3 秒后向上滑出并淡出。

练习 3：测试你的动画在"减少动态效果"的系统设置下是否仍然可用（macOS：系统偏好设置 → 辅助功能 → 显示 → 减少动态效果）。
