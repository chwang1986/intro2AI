03 控制流

条件判断与循环，让程序拥有决策能力

场景程序需要做出选择和重复

想象你每天早上出门前的决策过程：如果下雨，就带伞；如果温度低于 10 度，就穿羽绒服；否则穿普通外套。这不是直线式的「做 A，然后做 B」，而是根据条件选择不同的路径。

程序也需要这种能力。一个购物网站要判断「用户是否登录」来决定显示登录页还是个人中心；一个数据处理脚本要「重复读取每一行」直到文件结束。没有控制流，程序就只能从上到下死板地执行，和计算器没什么区别。

控制流让程序拥有了「智商」——它能根据情况做判断，能重复执行任务，能跳过不必要的步骤。这是从「脚本」走向「程序」的关键一跃。

原理程序执行的三种基本结构

无论多么复杂的程序，其控制流都可以分解为三种基本结构的组合：顺序、分支和循环。这是计算机科学家早在 1966 年就证明的定理——结构化程序定理。

顺序结构：代码从上到下依次执行，就像按照菜谱一步一步做菜。这是默认的执行方式。

分支结构（if）：根据条件选择执行路径，就像走到岔路口看路牌决定往哪边走。Python 用 if、elif、else 实现分支。

循环结构（for / while）：重复执行某段代码，就像洗衣机按照设定程序反复转动。Python 提供了 for（遍历序列）和 while（条件为真时持续执行）两种循环。

图解

要点条件判断与循环

if 语句让程序做选择，for 和 while 让程序重复执行。break 可以立即跳出循环，continue 则跳过当前轮次直接进入下一轮。

Python 的 for 循环设计得非常直观：「对于列表中的每一个元素，做某事」。你不需要关心索引和边界，只需要描述「对每个 item 做什么」。这让代码读起来像英文句子一样自然。

代码示例
# if / elif / else 条件判断
score = 85

if score >= 90:
    print("优秀")
elif score >= 60:
    print("及格")      # 这一行会被执行
else:
    print("不及格")

# for 循环遍历列表
fruits = ["苹果", "香蕉", "橙子"]
for fruit in fruits:
    print(fruit)

# while 循环
n = 3
while n > 0:
    print(n)
    n -= 1             # 不要忘记更新条件变量！

# break 和 continue
for num in range(10):
    if num == 3:
        continue       # 跳过 3，继续下一轮
    if num == 7:
        break          # 遇到 7，直接结束循环
    print(num)         # 输出: 0, 1, 2, 4, 5, 6

错误初学者常犯的错误

忘记缩进：Python 用缩进来表示代码块，而不是大括号。if 体、循环体、函数体都必须有统一的缩进。一旦混用空格和 Tab，或者某一行少缩进了一格，就会抛出 IndentationError。

while 死循环：如果 while 的条件永远为真，循环就永远不会结束。比如 while n > 0: print(n) 却忘了写 n -= 1，程序会疯狂打印 3 直到你强制终止。写 while 循环时，一定要确保条件最终会变为假。

在循环中修改正在遍历的列表：下面这段代码的预期是删除所有偶数，但结果却是错的——因为删除元素后列表长度和索引都发生了变化，循环会跳过某些元素。正确做法是遍历副本 for num in nums[:]:，或者先收集要删除的元素。

错误示例
# 错误：在遍历列表时直接删除元素
nums = [1, 2, 3, 4]
for num in nums:
    if num % 2 == 0:
        nums.remove(num)   # 危险！会跳过元素
print(nums)                # 结果可能是 [1, 3] 或更奇怪的东西

# 正确：遍历列表的副本
nums = [1, 2, 3, 4]
for num in nums[:]:
    if num % 2 == 0:
        nums.remove(num)
print(nums)                # [1, 3]

练习动手试一试

练习 1：写一个程序判断某一年是否为闰年。闰年规则：能被 4 整除但不能被 100 整除，或者能被 400 整除。

练习 2：用 for 循环和 range() 打印 1 到 100 之间所有的奇数。

练习 3：用 while 循环实现一个简单的猜数字游戏：预设一个答案（比如 7），让用户反复输入猜测，直到猜对为止，每次提示「太大了」或「太小了」。

练习 1 参考答案
year = 2024
if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    print(f"{year} 是闰年")
else:
    print(f"{year} 不是闰年")
