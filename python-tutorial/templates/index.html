Python 编程简明教程

从零开始掌握 Python，为 AI 时代打下编程基础

纲领这个教程要达成什么

最终目标：读完这个教程后，你能够用 Python 编写完整的程序——理解变量与类型、掌握控制流和函数、操作数据结构、处理文件与异常，并能调用 AI API 与大型语言模型协作。

面向读者：完全零基础，或者只写过简单脚本的人。不需要任何编程经验，每一步都有生动比喻和图解。

学习路径：理解 Python 本质 → 掌握基础语法 → 学会流程控制 → 编写函数 → 操作数据结构 → 文件与异常处理 → 面向对象编程 → 调用 AI API → Prompt 工程 → AI 协作工作流 → 综合实战。

路径学习路线说明

第一阶段（基础语法）：step1 到 step3 帮助你理解 Python 的本质，掌握变量、类型和控制流。这是所有编程的根基，就像学习写作前先认字。

第二阶段（组织能力）：step4 到 step7 教你如何把代码组织得更好——函数让代码复用，数据结构让数据有序，面向对象让程序结构清晰。

第三阶段（AI 实战）：step8 到 step11 进入 AI 时代的关键技能。你将学会调用 AI API、设计 Prompt、建立高效的人机协作工作流，最终完成一个完整的项目。
