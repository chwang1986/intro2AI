函数

把重复代码打包成可复用的工具箱

场景为什么需要函数？

想象你是一名厨师，每天要做 100 杯珍珠奶茶。如果没有标准化流程，你每次都要想"先放多少茶、多少奶、煮多久珍珠"——既累又容易出错。于是你写了一张配方卡：只要给出杯数和甜度，就能自动产出一致的奶茶。

函数就是程序里的配方卡。同一段代码在多处使用时，与其复制粘贴十几遍，不如把它装进一个"有名字的盒子"，需要时喊一声名字就能用。

原理函数的工作机制

函数是一段有名字的代码块，遵循"输入 → 处理 → 输出"的流水线模式。你可以把它看作一台自动贩卖机：从投币口塞入原料（参数），机器内部加工（执行代码），最后从取货口拿出成品（返回值）。

当程序调用函数时，Python 会在内存中开辟一块临时空间（称为栈帧），存放局部变量；函数执行完毕后，这块空间自动释放，返回值传回调用处。

图解

要点函数的核心语法

def 关键字用来定义函数；参数是函数的"入口"，可以设定默认值；return 是函数的"出口"，没有 return 默认返回 None。变量在函数内部创建的是局部变量，只在函数内可见。

代码示例
# 1. 基本函数定义与调用
def greet(name):
    """向用户打招呼"""
    message = f"你好，{name}！"
    return message

print(greet("小明"))   # 输出: 你好，小明！


# 2. 默认参数：给参数设置默认值
def make_coffee(type="美式", sugar=0):
    return f"一杯{type}，{sugar}勺糖"

print(make_coffee())               # 一杯美式，0勺糖
print(make_coffee("拿铁", 2))      # 一杯拿铁，2勺糖
print(make_coffee(sugar=1))        # 一杯美式，1勺糖（关键字传参）


# 3. return 返回多个值（实际是元组）
def calc(a, b):
    return a + b, a - b, a * b

sum_val, diff, prod = calc(10, 3)
print(sum_val, diff, prod)   # 13 7 30


# 4. 作用域：局部变量 vs 全局变量
count = 0   # 全局变量

def increment():
    global count   # 声明使用全局变量
    count += 1
    local_var = "我只能在函数内使用"
    return count

increment()
print(count)   # 1

错误常见陷阱

1. 忘记 return：函数没有 return 时默认返回 None，新手常误以为会得到计算结果。

def add(a, b):
    result = a + b
    # 糟糕！忘记写 return result

print(add(2, 3))   # 输出 None，不是 5

2. 在函数外访问局部变量：局部变量随函数结束而销毁，外部无法访问。

def test():
    x = 10

test()
print(x)   # NameError: name 'x' is not defined

3. 默认参数用可变对象：默认参数在函数定义时只创建一次，使用列表或字典作为默认值会导致意外累积。

# 错误示范
def append_item(item, items=[]):
    items.append(item)
    return items

print(append_item(1))   # [1]
print(append_item(2))   # [1, 2]  —— 糟糕，列表记住了上次的结果！

# 正确做法
def append_item(item, items=None):
    if items is None:
        items = []
    items.append(item)
    return items

练习动手试一试

练习 1：写一个函数 calculate_bmi(weight, height)，接收体重（kg）和身高（m），返回 BMI 值和对应的分类（偏瘦/正常/超重/肥胖）。BMI = 体重 / 身高²。

练习 2：写一个函数 average(*numbers)，接收任意多个数字，返回它们的平均值。如果没有传入数字，返回 0。

练习 3：写一个函数 is_palindrome(s)，判断一个字符串是否为回文（正读反读相同，如 "level"、"上海自来水来自海上"）。

练习 1 参考答案
def calculate_bmi(weight, height):
    bmi = weight / (height ** 2)
    if bmi < 18.5:
        category = "偏瘦"
    elif bmi < 24:
        category = "正常"
    elif bmi < 28:
        category = "超重"
    else:
        category = "肥胖"
    return bmi, category

bmi, cat = calculate_bmi(70, 1.75)
print(f"BMI: {bmi:.1f}, 分类: {cat}")
