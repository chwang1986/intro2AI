Step 4 — 效率推导

等效电路、阻抗匹配、最大效率条件

场景

设计一个无线输电系统时，最核心的指标是传输效率：输入功率中有多少百分比真正到达了负载。效率取决于哪些参数？如何优化？这就需要建立等效电路模型，进行严格的数学推导。

原理

等效电路模型

将发射端和接收端分别建模为串联 RLC 回路：

  • 发射端：电阻 $R_1$、电感 $L_1$、电容 $C_1$，由电源 $V_1$ 驱动。
  • 接收端：电阻 $R_2$、电感 $L_2$、电容 $C_2$，连接负载 $R_L$。
  • 耦合：通过互感 $M$ 连接，耦合系数 $k = M / \sqrt{L_1 L_2}$。
[等效电路图：两个 RLC 回路通过互感 M 耦合]
发射端：V1 — R1 — L1 — C1
接收端：R2 — L2 — C2 — RL

回路方程（谐振条件下）

当两个回路都调谐到同一谐振频率 $\omega_0$ 时，$j\omega L + \frac{1}{j\omega C} = 0$，电抗项抵消。回路方程简化为：

$$V_1 = I_1 R_1 - j\omega_0 M I_2$$ $$0 = I_2 (R_2 + R_L) - j\omega_0 M I_1$$

注意：这里 $j\omega_0 M I_2$ 是接收端电流在发射端感应的电动势（反射阻抗），$j\omega_0 M I_1$ 是发射端在接收端感应的电动势。

求解电流

从第二个方程：

$$I_2 = \frac{j\omega_0 M I_1}{R_2 + R_L}$$

代入第一个方程：

$$V_1 = I_1 R_1 - j\omega_0 M \cdot \frac{j\omega_0 M I_1}{R_2 + R_L} = I_1 \left( R_1 + \frac{(\omega_0 M)^2}{R_2 + R_L} \right)$$

因此发射端电流为：

$$I_1 = \frac{V_1}{R_1 + \frac{(\omega_0 M)^2}{R_2 + R_L}}$$

反射阻抗：$Z_r = \frac{(\omega_0 M)^2}{R_2 + R_L}$。接收端的存在，相当于在发射端增加了一个等效电阻。$Z_r$ 越大，说明能量传递到接收端越多。

效率公式推导

负载上的有用功率：

$$P_{out} = |I_2|^2 R_L = \frac{(\omega_0 M)^2 |I_1|^2 R_L}{(R_2 + R_L)^2}$$

发射端电源提供的总功率：

$$P_{in} = |I_1|^2 \left( R_1 + \frac{(\omega_0 M)^2}{R_2 + R_L} \right)$$

传输效率 $\eta = P_{out} / P_{in}$：

$$\eta = \frac{\frac{(\omega_0 M)^2 R_L}{(R_2 + R_L)^2}}{R_1 + \frac{(\omega_0 M)^2}{R_2 + R_L}} = \frac{(\omega_0 M)^2 R_L}{(R_2 + R_L)^2 R_1 + (\omega_0 M)^2 (R_2 + R_L)}$$

分子分母同除以 $(R_2 + R_L)$：

$$\eta = \frac{(\omega_0 M)^2 \frac{R_L}{R_2 + R_L}}{(R_2 + R_L) R_1 + (\omega_0 M)^2}$$

引入品质因数和耦合系数

定义发射端和接收端的品质因数：

$$Q_1 = \frac{\omega_0 L_1}{R_1}, \quad Q_2 = \frac{\omega_0 L_2}{R_2}$$

利用 $M = k\sqrt{L_1 L_2}$，有 $(\omega_0 M)^2 = k^2 \omega_0^2 L_1 L_2 = k^2 Q_1 Q_2 R_1 R_2$。

假设 $L_1 = L_2 = L$，$R_1 = R_2 = R$（对称系统），$Q_1 = Q_2 = Q$，则：

$$\eta = \frac{k^2 Q^2 \frac{R_L}{R + R_L}}{(R + R_L) R / R^2 + k^2 Q^2} \cdot \frac{R_L}{R + R_L}$$

经过代数整理（过程略），当负载与接收端线圈电阻匹配时（$R_L = R_2$，即负载电阻等于线圈电阻），效率达到最优：

$$\eta_{max} = \frac{k^2 Q^2}{(1 + \sqrt{1 + k^2 Q^2})^2}$$

更常用的简化形式是：

$$\eta = \frac{k^2 Q_1 Q_2}{(1 + \sqrt{1 + k^2 Q_1 Q_2})^2}$$

最大效率条件的直观理解

  • $k$ 越大越好：耦合越紧密，能量传递越直接。
  • $Q$ 越大越好：回路损耗越小，谐振峰越尖锐，能量在回路中"存储"的时间越长，有更多机会传递到另一端。
  • $kQ$ 是综合指标：实际工程中常用 $kQ$ 作为系统设计的核心参数。$kQ > 1$ 时，系统才能实现高效率（>50%）。
要点
  1. 谐振条件下效率最高：如果发射端或接收端没有调谐到谐振频率，电抗项不会抵消，额外的无功功率会降低效率。
  2. 阻抗匹配很重要：负载电阻 $R_L$ 应该与接收端线圈的电阻 $R_2$ 相匹配。太大或太小都会降低效率。
  3. $kQ$ 乘积决定效率上限：当 $kQ \gg 1$ 时，$\eta_{max} \approx 1 - 2/(kQ)$，趋近于 100%。
  4. 实际系统的限制：高 $Q$ 意味着低损耗电阻，需要粗导线或超导；高 $k$ 意味着近距离或大线圈。两者往往不能同时满足。
常见错误

❌ 忽略线圈电阻的影响
推导中 $R_1$ 和 $R_2$ 是线圈的串联电阻（铜损），不是负载电阻。很多人只考虑负载，忽略了线圈本身的损耗。

❌ 认为效率只与距离有关
距离只影响 $k$。即使距离固定，通过提高 $Q$（更好的线圈）或优化阻抗匹配，效率也可以显著提升。

❌ 在非谐振条件下套用效率公式
上述效率公式的前提是严格谐振。如果频率偏移，公式中的电抗项不能忽略，实际效率会比公式计算的更低。

示例
效率计算

已知：$k = 0.3$，$Q_1 = Q_2 = 200$。

求：最大传输效率。

解：

$$k^2 Q_1 Q_2 = 0.3^2 \times 200 \times 200 = 3600$$ $$\eta_{max} = \frac{3600}{(1 + \sqrt{1 + 3600})^2} = \frac{3600}{(1 + 60.01)^2} = \frac{3600}{3721} \approx 96.8\%$$

这是一个非常理想的结果。实际中由于谐振频率偏移、阻抗失配、涡流损耗等，效率通常在 80% ~ 90%。

练习

练习 1：已知 $k = 0.1$，$Q = 100$，计算最大传输效率。如果 $Q$ 提高到 300（使用更好的线圈），效率变为多少？

练习 2：一个无线输电系统的线圈电阻 $R = 0.5\,\Omega$，负载电阻 $R_L = 10\,\Omega$。判断是否满足阻抗匹配条件。如果不满足，效率会比最优值低多少？

练习 3：证明当 $kQ \gg 1$ 时，$\eta_{max} \approx 1 - \frac{2}{kQ}$。（提示：使用泰勒展开 $\sqrt{1+x} \approx \sqrt{x} + \frac{1}{2\sqrt{x}}$）
