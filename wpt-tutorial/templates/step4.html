Step 4 — 效率推导

等效电路、阻抗匹配、最大效率条件

场景

设计一个无线输电系统时，最核心的指标是传输效率：输入功率中有多少百分比真正到达了负载。效率取决于哪些参数？如何优化？这就需要建立等效电路模型，进行严格的数学推导。

起源从电路定律到效率公式

我们今天推导效率所用的工具——基尔霍夫电压定律（KVL）和基尔霍夫电流定律（KCL）——诞生于 19 世纪中叶。德国物理学家古斯塔夫·基尔霍夫（Gustav Kirchhoff）在 1845 年还是大学生时就提出了这两条定律，为电路分析奠定了数学基础。

到了 20 世纪初，工程师们开始研究如何把一个复杂的电路"简化"成更容易分析的形式。戴维南等效和诺顿等效定理告诉我们：任何线性有源二端网络，都可以等效为一个电压源串联一个电阻，或者一个电流源并联一个电阻。这种"等效变换"的思想，正是我们今天把无线输电系统建模为两个 RLC 回路的基础。

2007 年 MIT 团队在发表磁共振耦合论文时，首次将这一套成熟的电路分析方法系统地应用到了无线输电领域。他们证明：通过等效电路模型，可以精确预测传输效率，并找到最优设计参数。这篇论文不仅是一个实验报告，更是一套完整的工程设计方法论。

思考工程师如何思考效率问题？

假设你是一个无线输电系统的工程师。你的老板给你一组指标：传输距离 20cm，功率 3kW，效率不低于 90%。你从哪里入手？

第一步，你需要知道效率由什么决定。直觉告诉你：距离越近效率越高，线圈越大效率越高，导线越粗效率越高。但这些只是定性的感觉，你需要定量的公式来指导设计。

第二步，你建立一个简化模型。把发射端和接收端各看成一个"黑盒子"，里面包含电阻、电感和电容。两个黑盒子之间通过互感连接。这样，整个系统就变成了一张"电路图"——而电路图是可以用数学方程描述的。

第三步，你写出回路方程，求解电流，计算功率，最终得到效率的表达式。这个表达式会告诉你：在给定距离（即给定耦合系数 $k$）的情况下，需要多大的品质因数 $Q$ 才能达到 90% 的效率。如果 $Q$ 的要求太高（意味着需要超导或极粗的导线），你就知道这个指标在当前技术下不现实，需要和老板谈判。

这就是工程思维：把模糊的需求转化为数学模型，用模型推导出设计参数，再用参数判断可行性。没有推导，你就只能在黑暗中试错。

原理

等效电路模型

将发射端和接收端分别建模为串联 RLC 回路：

  • 发射端：电阻 $R_1$、电感 $L_1$、电容 $C_1$，由电源 $V_1$ 驱动。
  • 接收端：电阻 $R_2$、电感 $L_2$、电容 $C_2$，连接负载 $R_L$。
  • 耦合：通过互感 $M$ 连接，耦合系数 $k = M / \sqrt{L_1 L_2}$。
[等效电路图：两个 RLC 回路通过互感 M 耦合]
发射端：V1 — R1 — L1 — C1
接收端：R2 — L2 — C2 — RL

回路方程（谐振条件下）

当两个回路都调谐到同一谐振频率 $\omega_0$ 时，$j\omega L + \frac{1}{j\omega C} = 0$，电抗项抵消。回路方程简化为：

$$V_1 = I_1 R_1 - j\omega_0 M I_2$$ $$0 = I_2 (R_2 + R_L) - j\omega_0 M I_1$$

注意：这里 $j\omega_0 M I_2$ 是接收端电流在发射端感应的电动势（反射阻抗），$j\omega_0 M I_1$ 是发射端在接收端感应的电动势。

求解电流

从第二个方程：

$$I_2 = \frac{j\omega_0 M I_1}{R_2 + R_L}$$

代入第一个方程：

$$V_1 = I_1 R_1 - j\omega_0 M \cdot \frac{j\omega_0 M I_1}{R_2 + R_L} = I_1 \left( R_1 + \frac{(\omega_0 M)^2}{R_2 + R_L} \right)$$

因此发射端电流为：

$$I_1 = \frac{V_1}{R_1 + \frac{(\omega_0 M)^2}{R_2 + R_L}}$$

反射阻抗：$Z_r = \frac{(\omega_0 M)^2}{R_2 + R_L}$。接收端的存在，相当于在发射端增加了一个等效电阻。$Z_r$ 越大，说明能量传递到接收端越多。

效率公式推导

负载上的有用功率：

$$P_{out} = |I_2|^2 R_L = \frac{(\omega_0 M)^2 |I_1|^2 R_L}{(R_2 + R_L)^2}$$

发射端电源提供的总功率：

$$P_{in} = |I_1|^2 \left( R_1 + \frac{(\omega_0 M)^2}{R_2 + R_L} \right)$$

传输效率 $\eta = P_{out} / P_{in}$：

$$\eta = \frac{\frac{(\omega_0 M)^2 R_L}{(R_2 + R_L)^2}}{R_1 + \frac{(\omega_0 M)^2}{R_2 + R_L}} = \frac{(\omega_0 M)^2 R_L}{(R_2 + R_L)^2 R_1 + (\omega_0 M)^2 (R_2 + R_L)}$$

分子分母同除以 $(R_2 + R_L)$：

$$\eta = \frac{(\omega_0 M)^2 \frac{R_L}{R_2 + R_L}}{(R_2 + R_L) R_1 + (\omega_0 M)^2}$$

引入品质因数和耦合系数

定义发射端和接收端的品质因数：

$$Q_1 = \frac{\omega_0 L_1}{R_1}, \quad Q_2 = \frac{\omega_0 L_2}{R_2}$$

利用 $M = k\sqrt{L_1 L_2}$，有 $(\omega_0 M)^2 = k^2 \omega_0^2 L_1 L_2 = k^2 Q_1 Q_2 R_1 R_2$。

假设 $L_1 = L_2 = L$，$R_1 = R_2 = R$（对称系统），$Q_1 = Q_2 = Q$，则：

$$\eta = \frac{k^2 Q^2 \frac{R_L}{R + R_L}}{(R + R_L) R / R^2 + k^2 Q^2} \cdot \frac{R_L}{R + R_L}$$

经过代数整理（过程略），当负载与接收端线圈电阻匹配时（$R_L = R_2$，即负载电阻等于线圈电阻），效率达到最优：

$$\eta_{max} = \frac{k^2 Q^2}{(1 + \sqrt{1 + k^2 Q^2})^2}$$

更常用的简化形式是：

$$\eta = \frac{k^2 Q_1 Q_2}{(1 + \sqrt{1 + k^2 Q_1 Q_2})^2}$$

最大效率条件的直观理解

  • $k$ 越大越好：耦合越紧密，能量传递越直接。
  • $Q$ 越大越好：回路损耗越小，谐振峰越尖锐，能量在回路中"存储"的时间越长，有更多机会传递到另一端。
  • $kQ$ 是综合指标：实际工程中常用 $kQ$ 作为系统设计的核心参数。$kQ > 1$ 时，系统才能实现高效率（>50%）。
要点
  1. 谐振条件下效率最高：如果发射端或接收端没有调谐到谐振频率，电抗项不会抵消，额外的无功功率会降低效率。
  2. 阻抗匹配很重要：负载电阻 $R_L$ 应该与接收端线圈的电阻 $R_2$ 相匹配。太大或太小都会降低效率。
  3. $kQ$ 乘积决定效率上限：当 $kQ \gg 1$ 时，$\eta_{max} \approx 1 - 2/(kQ)$，趋近于 100%。
  4. 实际系统的限制：高 $Q$ 意味着低损耗电阻，需要粗导线或超导；高 $k$ 意味着近距离或大线圈。两者往往不能同时满足。

论证推导为什么是对的？

我们刚才的推导基于几个关键假设。如果这些假设不成立，公式就不可靠。让我们来审视这些假设的合理性。

假设 1：系统可以等效为两个 RLC 回路

这个假设在工程上是非常合理的。任何实际的线圈都有电阻（铜线损耗）、电感（磁场储能）和分布电容。在特定频率附近，分布电容可以等效为一个集中的电容 $C$，于是线圈就变成了 RLC 回路。这个近似在射频工程中已经使用了上百年，被无数实验验证。

假设 2：互感 $M$ 是唯一的耦合机制

在近距离（几米以内）且工作频率不高（几 MHz 以下）时，磁场耦合是主要的能量传递方式。电场耦合和电磁辐射的能量占比非常小，可以忽略。但如果频率升高到 GHz 级别，或者距离远到几十米，这个假设就不成立了——此时必须考虑电磁波辐射。

假设 3：阻抗匹配时效率最优

这个结论可以用微积分严格证明：对效率公式 $\eta$ 关于负载电阻 $R_L$ 求导，令导数为零，就能解出最优负载 $R_L = R_2$。这个结果和电路理论中的"最大功率传输定理"完全一致：当负载阻抗等于源阻抗的共轭时，传输功率最大。

核心论证：我们的推导不是"拍脑袋"编出来的公式，而是基于经过百年验证的电路定律（KVL/KCL）和数学优化方法。只要系统的物理结构符合模型假设，公式预测的结果就与实际测量高度吻合。这正是工程建模的价值所在。

如果公式错了怎么办？

科学的精髓在于可证伪性。如果实验测量的效率与公式预测严重不符，我们需要检查：是否严格谐振？线圈电阻是否被低估？是否存在额外的损耗机制（如涡流、铁芯损耗）？每一次"不符"都是发现新物理的机会。

常见错误

❌ 忽略线圈电阻的影响
推导中 $R_1$ 和 $R_2$ 是线圈的串联电阻（铜损），不是负载电阻。很多人只考虑负载，忽略了线圈本身的损耗。

❌ 认为效率只与距离有关
距离只影响 $k$。即使距离固定，通过提高 $Q$（更好的线圈）或优化阻抗匹配，效率也可以显著提升。

❌ 在非谐振条件下套用效率公式
上述效率公式的前提是严格谐振。如果频率偏移，公式中的电抗项不能忽略，实际效率会比公式计算的更低。

示例
效率计算

已知：$k = 0.3$，$Q_1 = Q_2 = 200$。

求：最大传输效率。

解：

$$k^2 Q_1 Q_2 = 0.3^2 \times 200 \times 200 = 3600$$ $$\eta_{max} = \frac{3600}{(1 + \sqrt{1 + 3600})^2} = \frac{3600}{(1 + 60.01)^2} = \frac{3600}{3721} \approx 96.8\%$$

这是一个非常理想的结果。实际中由于谐振频率偏移、阻抗失配、涡流损耗等，效率通常在 80% ~ 90%。

实验测量你的无线充电器效率

你不需要专业的功率分析仪，用一个支持功率显示的 USB 测试仪（约 20 元）和一个无线充电板，就能估算实际效率。

实验 1：估算手机无线充电效率

  1. 准备一个支持电压/电流显示的 USB 功率计（插在充电头和充电线之间）。
  2. 记录充电板空载时的输入功率 $P_{in,idle}$（通常为 0.5~2W，是充电板自身电路的待机功耗）。
  3. 把手机放在充电板上，记录此时的输入功率 $P_{in,total}$。
  4. 在手机设置中查看当前的无线充电功率 $P_{out}$（部分安卓手机支持显示，iPhone 需要借助第三方 App 或估算）。
  5. 计算近似效率：$\eta \approx \frac{P_{out}}{P_{in,total} - P_{in,idle}} \times 100\%$。

预期现象：手机无线充电的效率通常在 70%~80% 之间。你会注意到充电板和手机都会发热——发热的部分就是损失的功率。

实验 2：距离对效率的影响（定性）

  1. 在手机和充电板之间插入不同厚度的物体（如书本），每次增加 2mm。
  2. 观察 USB 功率计上的输入功率变化。如果输入功率下降，说明传输效率在降低（因为手机要求的充电功率相对固定，系统试图补偿但受限于耦合系数）。
  3. 当充电断开时，记录此时的厚度。

预期现象：随着距离增加，充电板需要消耗更多的电来维持相同的输出功率，但效率不断下降，最终无法维持充电。

实验 3：线圈对准对效率的影响

  1. 手机平放在充电板正中央，记录功率计读数。
  2. 横向移动手机，观察功率计读数变化。
  3. 尝试斜放手机，观察功率变化。

预期现象：横向偏移会降低耦合系数 $k$，从而降低效率，表现为功率计读数上升（系统试图补偿）或充电功率下降。

练习

练习 1：已知 $k = 0.1$，$Q = 100$，计算最大传输效率。如果 $Q$ 提高到 300（使用更好的线圈），效率变为多少？

练习 2：一个无线输电系统的线圈电阻 $R = 0.5\,\Omega$，负载电阻 $R_L = 10\,\Omega$。判断是否满足阻抗匹配条件。如果不满足，效率会比最优值低多少？

练习 3：证明当 $kQ \gg 1$ 时，$\eta_{max} \approx 1 - \frac{2}{kQ}$。（提示：使用泰勒展开 $\sqrt{1+x} \approx \sqrt{x} + \frac{1}{2\sqrt{x}}$）

练习 4（设计题）：某电动汽车无线充电系统要求效率 >92%，传输距离 25cm（对应 $k \approx 0.25$）。请计算所需的最低品质因数 $Q$（假设对称系统 $Q_1 = Q_2 = Q$），并讨论实现这个 $Q$ 值的可行性。

练习 5（验证题）：查阅 Litz 线（利兹线）的资料，解释为什么高频无线输电线圈使用 Litz 线而不是普通铜线。这与效率公式中的哪个参数有关？
